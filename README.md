# NyayaAI – AI Legal Assistant for Indian Citizens

A modern, responsive frontend for a legal‑tech platform.

## Features
- Landing page with hero, features, testimonials, FAQ
- AI Assistant demo with simulated chat
- Legal Guides page
- Downloadable legal templates
- Login / Sign Up interface

## How to Run
Open `index.html` in any browser, or use Live Server in VS Code.

## Tech Stack
HTML, CSS, JavaScript (vanilla)
